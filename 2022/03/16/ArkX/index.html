ArkX 明日方舟自动化助手

基于 TensorFlow 与 Scrcpy 的开源自动化明日方舟清理智工具

项目地址

Github

视频教程

bilibili

项目功能

已经实现的功能(核心功能)

    • 对游戏界面进行判断(实验结论为使用自定义网络的Tensorflow的图像分类模型)
    • 从图像中获取各类按钮的坐标(实验结论为使用基于Tensorflow的ObjectDetectionAPI库)
    • 识别文字(实验结论为使用同样基于Tensorflow的PaddleOCR)
    • 利用adb获取串流(经过实验后结论为使用 scrcpy)
    • 利用adb实现点击(也使用scrcpy)
    • 用于清理智的操作逻辑程序

其他重要功能

  • 自动添加理智
  • 判断是否使用源石添加理智
  • 读取进度
  • 反馈模型检测精准度
  • 降低CPU使用率
  • 支持GPU
  • 设置GPU显存限制
  • GPU动态显存
  • 多设备选择

已实现

  • 提供adb设备操作界面
  • 显示系统信息(主要为GPU设备)
  • 提供网络设备的连接功能

待实现

  • 基于ffmpeg-python 对 H264解码进行GPU加速支持(cuvid,cuda,h264_amf,vaapi), 移除PyAV库

大饼

  • 实现基建自动换班
  • 实现指定不同关卡
