ArkRecord

本文最后更新于：2022年4月15日 下午

这是我写的一个Chrome浏览器的插件

用于记录明日方舟的抽卡记录，并且展示统计数据，支持导出导入(暂时只支持json数据)

项目地址

GitHub repository

视频教程

Bilibili

安装方法

方法一

Chrome Store安装

还在等Google的审核

后续补齐

Google 那边已经过审了

在Chrome Web Store中搜索 ArkRecord 即可安装

或者直接点击 https://chrome.google.com/webstore/detail/arkrecord/mphjjhidfbbnjjnmpffepbfllgocggpl 安装

然后跳到使用方法

方法二

下载release版本

  1. 前往 Release Page 下载Packaged.arkrecord.zip

  2. 将Packaged.arkrecord.zip解压到一个你认为合适的地方

  3. 接下来打开Chrome浏览器，在地址栏中输入chrome://extensions并回车

  4. 把右上角的 开发者模式 打开

  5. 再点击左上角的 加载已解压的扩展程序, 选择 Packaged.arkrecord.zip 解压后所在的目录

使用方法

  1. ⚠️⚠️⚠️(重点) 使用插件之前，先要确保你登录了明日方舟的官网 https://ak.hypergryph.com/ ,因为插件是基于方舟官网的cookie进行记录的

  2. 前往右上角的插件图标 找到 ArkRecord 的图标

  3. 然后可以自行选择是否固定(点击右侧的图钉图标可以固定住插件)

  4. 点击图标即可打开插件的缩略信息页面, 如果想要看详细数据, 或者进行数据导入导出, 只需要点击 查看详细数据 按钮

更新日志

2022/04/15

增加了两项统计数据的显示: 数量统计 与 去重统计

数量统计 代表这个表统计范围内该类型干员们总共被抽出来多少个

去重统计 代表的是这个表格统计范围内，抽出来多少名干员，也就是说一位干员被重复抽出，重复部分不会被记录进这项数据
