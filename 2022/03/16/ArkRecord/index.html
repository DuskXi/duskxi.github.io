ArkRecord

本文最后更新于：2022年7月19日 晚上

这是我写的一个Chrome浏览器的插件

用于记录明日方舟的抽卡记录，并且展示统计数据，支持导出导入(暂时只支持json数据)

安装方法

方法一

Chrome Store安装

还在等Google的审核

后续补齐

Google 那边已经过审了

在Chrome Web Store中搜索 ArkRecord 即可安装

或者直接点击 https://chrome.google.com/webstore/detail/arkrecord/mphjjhidfbbnjjnmpffepbfllgocggpl 安装

然后跳到使用方法

方法二

下载release版本

  1. 前往 Release Page 下载Packaged.arkrecord.zip

  2. 将Packaged.arkrecord.zip解压到一个你认为合适的地方

  3. 接下来打开Chrome浏览器，在地址栏中输入chrome://extensions并回车

  4. 把右上角的 开发者模式 打开

  5. 再点击左上角的 加载已解压的扩展程序, 选择 Packaged.arkrecord.zip 解压后所在的目录

使用方法

  1. ⚠️⚠️⚠️(重点) 使用插件之前，先要确保你登录了明日方舟的官网 https://ak.hypergryph.com/ ,因为插件是基于方舟官网的cookie进行记录的。现在已经加入对B服数据的支持，在 https://ak.hypergryph.com/user/login 中选择”使用Bilibili账号”登录即可。

  2. 前往右上角的插件图标 找到 ArkRecord 的图标

  3. 然后可以自行选择是否固定(点击右侧的图钉图标可以固定住插件)

  4. 点击图标即可打开插件的缩略信息页面, 如果想要看详细数据, 或者进行数据导入导出, 只需要点击 查看详细数据 按钮

更新日志

2022/04/15 v0.0.0.3

增加了两项统计数据的显示: 数量统计 与 去重统计

数量统计 代表这个表统计范围内该类型干员们总共被抽出来多少个

去重统计 代表的是这个表格统计范围内，抽出来多少名干员，也就是说一位干员被重复抽出，重复部分不会被记录进这项数据

2022/07/07 v0.0.0.4

增加了对常驻标准寻访的独立统计

2022/07/16 v0.0.0.5

普通更新内容:

  1. 表格鼠标悬停显示详细干员名单, 并且提供开关切换是否在详细名单中统计数量
  2. 纯常驻寻访统计加入合并统计功能，因为在实际抽卡情况中有可能会出现玩家抽了几次标准寻访后再去抽活动卡池，这样就会导致数据被隔断，所以提供合并统计功能。只需要用户点击对应的复选框即可
  3. 新增 图表 页面，这个页面提供了三种图表分析：线图(时间-各项干员数量)、柱状图(卡池出货比例)、饼图(合并统计卡池比例，可选择加入什么卡池，饼图有两张，一张全数据，一张去重数据)
  4. 新增 统计学分析 页面，这个页面将会提供抽卡数据欧非分析，会显示全部数据的欧非程度，30天近期欧非程度，以及提供每一个抽卡阶段（抽卡阶段指每个指定的干员（6或5）抽到之前垫了多少抽）的欧非程度分析和按照卡池统计的欧非程度

重要更新内容:

对B服数据的支持, 只需要在 https://ak.hypergryph.com/user/login 中选择”使用Bilibili账号”登录后，在ArkRecord的Popup页面或者分析页面点击右下角的刷新数据或者直接重启浏览器即可更新数据
